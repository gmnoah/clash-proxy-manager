# 代理控制器

## 项目概述

Clash Verge 代理控制器 - 一个桌面应用，用于一键开关汽水音乐、B站等代理服务。

将原来的两个 `.command` 脚本（开启/关闭汽水音乐代理）整合为带GUI界面的桌面应用。

## 主要功能

- 可视化界面管理代理服务
- 一键开启/关闭代理
- 支持多个代理服务（汽水音乐、B站等）
- 配置持久化存储

## 技术栈

| 技术 | 用途 |
|-----|------|
| Tauri 2 | 桌面应用框架 |
| Vue 3 | 前端框架 |
| TypeScript | 类型安全 |
| Vite | 构建工具 |
| Element Plus | UI 组件库 |

## 运行命令

```bash
# 开发模式
./run.sh

# 构建打包
./build.sh

# 清理缓存
./clean.sh -y
```

## 配置文件

配置文件位于 `~/.config/proxy-controller/services.json`：

```json
{
  "services": [
    {
      "id": "qishui",
      "name": "汽水音乐",
      "icon": "🎵",
      "scriptPath": "/path/to/qishui-proxy-control.sh",
      "enabled": false
    }
  ]
}
```

### 添加新的代理服务

1. 编辑配置文件
2. 添加新的服务条目：
   ```json
   {
     "id": "bilibili",
     "name": "哔哩哔哩",
     "icon": "📺",
     "scriptPath": "/path/to/bilibili-proxy-control.sh",
     "enabled": false
   }
   ```
3. 点击刷新按钮或重启应用

## 备注

- 脚本需要支持 `enable` 和 `disable` 两个参数
- 脚本路径必须是绝对路径
- 首次启动会自动创建默认配置文件
- 原来的 `.command` 文件已移至 `archive/` 目录
